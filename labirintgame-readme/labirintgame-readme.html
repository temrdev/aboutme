Labirint Game - READ ME

Description: A maze game made with Python and pygame.

Your purpose to move through walls, find enemies and kill them. Your gift will be a pacman icon :3

If you win the game, you will see a winning screen with a like!

If you lose the game, you will see a game over screen

Features

  • Player movement using arrow keys
  • Collisions and obstacles
  • Win condition

Installation

To run the game, you need to have Python and pygame installed. You can install pygame using pip:

pip install pygame
