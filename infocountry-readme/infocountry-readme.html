InfoCountry - README

Description: A country information app made with Python and PyQt5.

Features

  • Interaction with country data (add,edit,delete)
  • Ergonomic user interface
  • Responsive design
  • Database integration

Installation

To run the app, you need to have Python and PyQt5 installed. You can install PyQt5 using pip:

pip install PyQt5

Detailed guide how to execute the app:

  1. Clone the repository: git clone https://github.com/temrdev/infocountry.git
  2. Navigate to the project directory: cd infocountry
  3. Run the app: python main.py
